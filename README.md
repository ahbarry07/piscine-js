## Overview

The JavaScript Piscine ran from **September 18 to October 13, 2023** and was successfully completed. Structured around the 42 School methodology, it covered JavaScript from the ground up — progressively introducing language concepts, browser APIs, Node.js, and asynchronous patterns through a series of graded projects.

## What I Learned

### JavaScript Core
- Variables, scoping (`var`, `let`, `const`) and closures
- Functions as first-class citizens, callbacks, higher-order functions
- Prototype chain and object-oriented patterns
- Iterators, generators, and array methods (`map`, `filter`, `reduce`)

### Asynchronous JavaScript
- Event loop and concurrency model
- Callbacks → Promises → `async/await` progression
- Fetch API and HTTP request handling
- Error handling in async contexts

### Browser & DOM
- DOM traversal, manipulation, and event handling
- Dynamic rendering and UI interactivity
- Browser APIs (timers, storage)

### Node.js
- CommonJS module system
- File system operations (`fs`)
- Building CLI tools with Node

### Applied Projects
- **CLONERNEWS** — A functional clone of Hacker News built using the HN API, demonstrating async data fetching and dynamic DOM rendering
- **SORTABLE** — An interactive drag-and-drop sortable list UI
- **DOM** — Full DOM manipulation exercises simulating real-world UI components
